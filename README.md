# Personal Diary

SPA web-приложение для заметок с авторизацией

## Используемые библиотеки
- RxJs
- ngx-editor
- primeng
- Firebase Auth
- Realtime Firebase 

## Запуск

`npm install` - установка библиотек.
`ng serve --open` - запуск приложения.
